---

## 기능

| 기능 | 설명 |
|------|------|
| 저장 | 사이트 이름, 아이디, 비밀번호 저장 (카테고리·메모·URL 포함) |
| 검색 | 사이트 이름으로 실시간 검색 + 카테고리 필터 |
| 복사 | 아이디·비밀번호 옆 "복사" 버튼으로 클립보드 복사 |
| 유출 조회 | 이메일로 개인정보 유출 사이트 조회 |
| PIN 잠금 | 4자리 PIN으로 앱 잠금 |
| 비밀번호 강도 | 입력 중 강도 표시 + 자동 생성 기능 |
| 보안 점검 | 약한/중복/오래된 비밀번호 분석 리포트 |
| 백업 | JSON 파일로 내보내기/불러오기 |
| 다크모드 | 어두운 화면 전환 |
| 글씨 크기 | A− / A＋ 버튼으로 4단계 조절 |

---

> 데이터는 내 브라우저에만 저장되며, 외부 서버로 전송되지 않습니다.
