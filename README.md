# 🔐 비밀번호 메모장 (Passecure)

## ▶ 클릭하면 바로 실행됩니다

**[👉 비밀번호 메모장 열기](https://htmlpreview.github.io/?https://github.com/lgw7126/Passecure/blob/claude/password-memo-webapp-emrdep/index.html)**

> 위 링크를 클릭하면 설치 없이 브라우저에서 바로 사용할 수 있습니다.

---

## 기능

| 기능 | 설명 |
|------|------|
| 저장 | 사이트 이름, 아이디, 비밀번호를 입력하고 저장 |
| 검색 | 사이트 이름으로 실시간 검색 |
| 복사 | 아이디·비밀번호 옆 "복사" 버튼으로 클립보드 복사 |
| 데이터 유지 | 브라우저를 껐다 켜도 데이터 그대로 유지 |

## 사용 방법

1. 위 링크를 클릭해서 앱을 엽니다.
2. 사이트 이름, 아이디, 비밀번호를 입력하고 **저장하기** 버튼을 누릅니다.
3. 저장된 목록에서 **복사** 버튼을 눌러 클립보드에 바로 복사합니다.
4. 검색창에 사이트 이름을 입력하면 해당 항목만 보입니다.

## 직접 실행 (파일 다운로드)

인터넷 없이 사용하려면 `index.html` 파일을 다운로드해서 더블클릭으로 여세요.

---

> 데이터는 내 브라우저에만 저장되며, 외부 서버로 전송되지 않습니다.
