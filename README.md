# 🔐 비밀번호 메모장 (Passecure)

## ▶ 클릭하면 바로 실행됩니다

**[👉 비밀번호 메모장 열기](https://lgw7126.github.io/Passecure/)**

> 위 링크를 클릭하면 설치 없이 브라우저에서 바로 사용할 수 있습니다.

---

## 소개

**Passecure**는 내 비밀번호를 안전하게 메모해두는 개인용 웹앱입니다.

앱 설치가 필요 없고, 모든 데이터는 **내 기기(브라우저)에만 저장**됩니다.  
외부 서버로 어떤 정보도 전송되지 않아 완전한 프라이버시를 보장합니다.

- 인터넷 없이도 사용 가능 (저장된 데이터는 오프라인에서도 유지)
- 4자리 PIN으로 앱을 잠가 다른 사람이 열 수 없게 보호
- PC·스마트폰·태블릿 모두 사용 가능

---

## 기능

| 기능 | 설명 |
|------|------|
| 저장 | 사이트 이름, 아이디, 비밀번호 저장 (카테고리·메모·URL 포함) |
| 검색 | 사이트 이름으로 실시간 검색 + 카테고리 필터 |
| 복사 | 아이디·비밀번호 옆 "복사" 버튼으로 클립보드 복사 |
| 유출 조회 | 이메일로 개인정보 유출 사이트 조회 |
| PIN 잠금 | 4자리 PIN으로 앱 잠금 |
| 비밀번호 강도 | 입력 중 강도 표시 + 자동 생성 기능 |
| 보안 점검 | 약한/중복/오래된 비밀번호 분석 리포트 |
| 백업 | JSON 파일로 내보내기/불러오기 |
| 다크모드 | 어두운 화면 전환 |
| 글씨 크기 | A− / A＋ 버튼으로 4단계 조절 |

---

## 사용 방법

1. 위 링크를 클릭해 앱을 엽니다
2. 처음 실행 시 PIN 번호(4자리)를 설정합니다
3. 사이트 이름·아이디·비밀번호를 입력하고 **저장하기**
4. 저장된 항목에서 **복사** 버튼으로 빠르게 붙여넣기

---

> 데이터는 내 브라우저에만 저장되며, 외부 서버로 전송되지 않습니다.
